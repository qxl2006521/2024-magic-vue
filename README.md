# 2024-magic-vue

在抖音看到渡一老师讲了2024春晚刘谦魔术的前端实现.由于没找到他的源码,就按照他的思路自己实现了一个.用的vue3,没用ts.

## 原教程
[渡一教程](https://www.douyin.com/video/7339410981212163363).

## 运行方法
 ```sh
npm install
npm run dev
```
